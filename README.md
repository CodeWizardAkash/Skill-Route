##üå± How to Contribute

There are many beginner-friendly ways to contribute to this project:

###Add new learning paths or topics
-Extend existing learning paths or create new ones by updating the data files. This is a great way to contribute without touching complex logic.

###Add project ideas
-Add new project ideas in src/data/projects.json. You can include a description, difficulty level, skills used, and an optional GitHub or live demo link.

###Improve UI and responsiveness
-Enhance the user interface, improve layouts, colors, spacing, or make the design more responsive across devices using Tailwind CSS.

###Improve documentation
-Help make the README, contribution guide, or comments clearer and more beginner-friendly.
